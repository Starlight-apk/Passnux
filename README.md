# Passnux 🚀

> **Pass**ion + Li**nux** = 为 ARM64 而生的 AI 公益站

## 🌟 项目简介

Passnux 是一个运行在 **ARM64 Linux** 架构上的 AI 公益服务平台。致力于让每个人都能免费、便捷地使用人工智能技术，打破算力壁垒，让 AI 真正普惠大众。

### ✨ 核心理念

- 🆓 **完全免费** — 公益为先，不设付费墙
- 🐧 **ARM64 优先** — 专为国产化、信创环境优化
- 🔓 **开源开放** — 代码透明，社区共建
- 🌐 **人人可用** — 降低 AI 使用门槛

## 🎯 项目目标

- [ ] 搭建 AI 对话服务（支持主流大模型 API）
- [ ] 提供多模型聚合访问
- [ ] 实现用户友好的 Web 界面
- [ ] 支持 ARM64 架构高效运行
- [ ] 持续集成与部署

## 📦 环境要求

- **操作系统**: Linux (ARM64 / aarch64)
- **Python**: >= 3.10
- **Node.js**: >= 18
- **内存**: >= 2GB（推荐 4GB+）
