# Usage FAQ

## AI Answers Are Inaccurate?

AI can sometimes make mistakes. Suggestions:
- Rephrase your question with more context
- Switch to a different model for comparison
- Ask multiple times for more accurate results

## Is There a Character Limit?

Each conversation has a maximum of 4096 tokens. After exceeding this, you need to start a new conversation.

## Does It Support File Uploads?

Currently, only text input is supported. File upload is not yet available.

## Why Is the Response Slow?

Possible reasons:
- High number of concurrent users
- Unstable network connection
- The model itself responds slowly

## Will the Service Always Be Free?

Passnux is a public welfare project and is currently committed to being free forever. Any changes will be announced in advance.
