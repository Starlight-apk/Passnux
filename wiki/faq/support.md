# 技术支持

## 遇到问题怎么办？

1. 先查看本 wiki 文档，看是否有解决方案
2. 检查网络连接是否正常
3. 尝试刷新页面或重新登录

## 反馈渠道

| 方式 | 说明 |
| :--- | :--- |
| GitHub Issues | 提交 Bug 或功能建议 |
| 邮件 | 发送至项目维护者邮箱 |
| 社区 | 加入用户交流群 |

## 报告 Bug

请提供以下信息：
- 问题描述和复现步骤
- 浏览器类型和版本
- 截图（如有）
- 出现时间

## 功能建议

欢迎提交功能建议！我们会根据用户反馈持续改进。
