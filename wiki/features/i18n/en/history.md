# Chat History

## Viewing History

All conversation records are automatically saved in the left sidebar. Click any conversation to view it.

## Conversation Management

| Action | Description |
| :--- | :--- |
| Rename | Right-click a conversation to edit its title |
| Delete | Delete unwanted conversation records |
| Search | Enter keywords in the search box to find conversations |
| Export | Export conversations as text files |

## Data Storage

- Conversation records are stored on the server
- Accessible from any device after logging in
- We recommend regularly exporting important conversations

## Privacy Notice

- Conversation data is encrypted
- Visible only to you
- Administrators cannot view your conversation content
