# AI Chat Feature

## Chat Modes

Passnux supports two chat modes:

### Normal Mode
Question-and-answer style. The AI remembers the context of the current conversation.

### Streaming Mode
The AI outputs responses word by word, showing the thinking process in real time for faster responses.

## Chat Operations

| Action | Description |
| :--- | :--- |
| Send Message | Press Enter after typing |
| Clear Chat | Delete all messages in the current conversation |
| Copy Reply | Click the copy button on the top-right of a reply |
| Regenerate | Ask the AI to re-answer the previous question |

## Notes

- Maximum of 4096 tokens per conversation
- Connection will automatically disconnect after prolonged inactivity
- Sensitive content will be filtered
