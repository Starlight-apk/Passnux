# Multi-Model Switching

## Supported Models

Passnux integrates multiple AI models that you can freely switch between:

| Model | Features | Use Case |
| :--- | :--- | :--- |
| DeepSeek | Fast, good at Chinese | Daily conversation |
| GPT-3.5 | Strong English capability | English writing |
| Tongyi Qianwen | Deep Chinese understanding | Chinese content creation |

## How to Switch

Click the model name at the top of the chat interface and select from the dropdown menu.

## Model Comparison

Different models may give different answers to the same question. Suggestions:

- Use DeepSeek for simple questions (fastest response)
- Use GPT-3.5 for English content
- Use Tongyi Qianwen for Chinese content creation
- For complex questions, switch models to compare answers
